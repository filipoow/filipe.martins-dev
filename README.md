# Filipe Martins - Currículo Interativo


## Sobre o Projeto

Este repositório contém o código-fonte do currículo interativo e portfólio. O projeto serve como um resumo visual e técnico da minha experiência em Engenharia de Dados, Automação, Machine Learning e Análise de Negócios.

O objetivo principal é demonstrar minha capacidade de transformar dados complexos em soluções que geram impacto real nas áreas comercial e de operações.
